 Линукс это прям MUST HAVE для каждого. Для быстрого въезжания в суть вопроса обычно советуют поставить любимый школьниками Kali Linux.

 Пока я пройдусь лишь по основным командам:

 PWD - эта команда выводит на экран полный путь к текущем каталогу. Используя её можно узнать где ты сейчас находишься, если заблудишься.

 ls - команда выводит список файлов и подкаталогов указанного каталога. Чтобы вывести файлы имена которых скрыты надо указать ключ -a
 ls -a
 А чтобы помимо имен увидеть и полную информацию о файлах, нужно добавить ключ -l. И у нас выйдет команда:
 ls -al

 Cat - команда cat позволяет вывести содержимое указанного файла. Чтобы вывести содержимое например test.txt надо выполнить следующую команду: cat test.txt

 Tac - команда обратная команде cat она выводит на экран файл в обратном порядке, начиная с последней строки до первой.

 cd - команда cd позволяет сменить текущий каталог. Для этого необходимо в качестве параметра задать нужную папку:
 cd /home/music

 cp - команда для копирования файла. С её помощью можно:
 	>>>Копировать содержимое файла в другой документ той же папки:
 	  cp /home/test/example1.txt /home/test/example2.txt
 	>>>Копировать файл в другой каталог:
 	  cp /home/test/example.txt /home/test2/example.txt
 	>>>Копировать несколько файлов в новый каталог:
 	  cp /home/test/example1.txt /home/test/example2.txt /home/neotest

 mkdir - Создание нового каталога. Например, чтобы создать новый подкаталог example в текущем каталоге, нужно выполнить команду:
 	$ mkdir example
 Маски:
 	-d - удалить каталог
 	-r - рекурсивно удалять содержимое каталогов
 	-f - не запрашивать подтверждение на удаление файлов

 df - этак команда позволяет узнать объем свободного места на жестком диске или разделе.
